# MASK MVP Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the first compiling MVP of `mask` as a panic-free, async-first LAN-discovered RAG service for shared code knowledge retrieval.

**Architecture:** The binary exposes two subcommands: `serve` builds an in-memory embedding index from a directory, announces itself through CAMP, and serves `POST /query` over Axum; `query` discovers a `knowledge-base` peer through CAMP, forwards the question with Reqwest, and prints only the JSON result array. The code stays split by responsibility so indexing, serving, output formatting, and CLI orchestration can evolve independently.

**Tech Stack:** Rust 2024, tokio, clap, axum, reqwest, serde, serde_json, fastembed, coding_agent_mesh_presence.
